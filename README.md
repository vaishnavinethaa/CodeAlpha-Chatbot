# 🤖 Smart Chatbot using Python

## Project Overview
This is a simple Smart Chatbot developed using Python as part of my CodeAlpha Internship.

## Features
- Greets the user
- Accepts user input
- Answers predefined questions
- Provides information about Python and AI
- Continuous conversation using loops
- Exit functionality

## Technologies Used
- Python
- VS Code

## Concepts Used
- Variables
- Input and Output
- If-Elif-Else Statements
- While Loop
- String Methods
